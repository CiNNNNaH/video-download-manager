# Package 1.5 Runtime Gate

This package is a regression-isolation pass focused on the analyze/render pipeline after the language-normalization work.

Key fixes:
- Advanced View becomes the default startup mode.
- Simple View remains available, but now builds a fuller representative ladder instead of collapsing to only Best Compatible and Audio Only on rich YouTube format inventories.
- The initial selected row avoids storyboard/mhtml entries.
- The selected-format summary is presented in a dedicated band below the format table, instead of visually blending into the table area.
