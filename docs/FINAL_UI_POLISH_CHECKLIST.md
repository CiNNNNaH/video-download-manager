# Final UI Polish Checklist

## Layout
- format table remains the dominant panel
- selected-format summary never overlaps any other widget
- status area remains readable with long dependency lines
- long filenames remain readable or safely wrapped

## Interaction
- column order and widths persist after restart
- cookie source / media mode / container / remux / fallback persist after restart
- stop action stays non-modal
- `Open File` opens the file, not Explorer
- `Open Folder` opens Explorer to the download folder

## Data presentation
- prefer `filesize` or `filesize_approx` over protocol notes
- keep protocol/technical info visible but secondary
- default column order should remain user-friendly
